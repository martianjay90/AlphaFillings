# 개발 서버 사용 가이드

## 포트 고정 설정

개발 서버는 항상 **포트 3000**으로 고정됩니다.

## 사용 방법

### 정상 시작
```bash
npm run dev
```
- 항상 `http://localhost:3000`에서 접속 가능합니다.

### 문제 발생 시 리셋
```bash
npm run dev:reset
```

이 명령은 다음을 수행합니다:
1. 포트 3000, 3001, 3002, 3003, 3004를 모두 종료
2. `.next` 캐시 폴더 삭제
3. 포트 3000으로 개발 서버 재시작

## 주의사항

- **항상 `http://localhost:3000`으로 접속하세요**
- 포트가 사용 중이면 `npm run dev:reset`을 실행하세요
- 더 이상 3003, 3004 등 다른 포트로 자동 변경되지 않습니다
